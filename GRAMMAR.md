## 1. 기본 규칙

### 들여쓰기

공백 4칸 또는 탭 1개. Python과 동일한 방식으로 블록을 구분합니다.

```han
만약 참:
    출력("들여쓰기 블록")
```

### 주석

```han
// 한 줄 주석

/*
   여러 줄 주석
*/
```

### 문장 구분

줄바꿈이 문장 구분자입니다. 세미콜론이 필요 없습니다.

### 식별자 규칙

한글·영문·숫자·밑줄(`_`) 모두 사용 가능합니다. 숫자로 시작할 수 없습니다.

```han
정수 나이 = 20
정수 user_age = 20
정수 나이2 = 20
```

## 2. 자료형

| 키워드 | 설명 | 예시 |
|--------|------|------|
| `정수` | 정수 (64비트) | `42`, `-7`, `0` |
| `실수` | 부동소수점 (64비트) | `3.14`, `-0.5` |
| `문자` | 문자열 (string) | `"안녕하세요"` |
| `글자` | 단일 문자 (char) | `'가'`, `'A'` |
| `논리` | 불리언 | `참`, `거짓` |
| `없음` | null / void | |
| `행렬` | 2D 배열 | `[[0.1, 0.2], [0.3, 0.4]]` |
| `텐서` | N차원 배열 | `텐서([1, 2, 3], [3])` |
| `사진` | 이미지 파일 핸들 | `사진열기("photo.jpg")` |
| `그림` | 캔버스 핸들 | `그림만들기(800, 600)` |
| `함수형` | 람다 변수 | `(x) => x * x` |

### 수 리터럴

```han
정수 십진수  = 255

실수 부동소수 = 3.14
실수 지수표기 = 1.5e10
```

### 문자열 이스케이프

```han
"\n"   // 줄바꿈
```

## 4. 연산자

### 산술

| 연산자 | 설명 | 예시 |
|--------|------|------|
| `+` | 덧셈 / 문자열 연결 | `1 + 2`, `"가" + "나"` |
| `-` | 뺄셈 | `5 - 3` |
| `*` | 곱셈 | `2 * 3` |
| `/` | 나눗셈 (실수) | `7 / 2` → `3.5` |
| `//` | 정수 나눗셈 | `7 // 2` → `3` |
| `%` | 나머지 | `7 % 2` → `1` |
| `**` | 거듭제곱 | `2 ** 8` → `256` |

### 비교

```han
== != < > <= >=
```

### 논리

```han
그리고    // and
또는      // or
아니다    // not
```

### 비트

```han
&   // AND
|   // OR
^   // XOR
~   // NOT
<<  // 왼쪽 시프트
>>  // 오른쪽 시프트
```

### 형변환

```han
정수(값)    실수(값)    문자(값)    글자(값)    논리(값)
```

## 5. 제어문

### 조건문

```han
만약 조건:
    // 참일 때
아니면 만약 다른조건:
    // 두 번째 조건 참일 때
아니면:
    // 모두 거짓일 때
```

### 선택문 (switch)

```han
선택 값:
    경우 1:
        출력("하나")
    경우 2:
        출력("둘")
    그외:
        출력("기타")
```

### 범위 반복

```han
반복 i 부터 1 까지 10:
    출력(i)       // 1, 2, ..., 10

반복 i 부터 10 까지 1:
    출력(i)       // 역순
```

### 조건 반복

```han
동안 조건:
    // 조건이 참인 동안 반복
    멈춤          // break
    건너뜀        // continue
```

### 순회 반복

```han
각각 항목 안에 배열:
    출력(항목)

각각 키, 값 안에 사전:
    출력(키 + ": " + 값)
```

### 흐름 제어

```han
멈춤          // 반복 종료 (break)
건너뜀        // 다음 순서로 (continue)
반환          // 함수 반환 (return)
끝냄          // 반환값 없이 종료 (return;)
이동 이름:    // goto
이름:         // 레이블
```

## 6. 함수

### 함수 (반환값 있음)

```han
함수 이름(매개변수들):
    // 본문
    반환 값
```

```han
함수 더하기(정수 가, 정수 나):
    반환 가 + 나

결과 = 더하기(3, 5)    // 8
```

### 정의 (반환값 없음)

```han
정의 이름(매개변수들):
    // 본문
```

```han
정의 인사(문자 이름):
    출력("안녕하세요, " + 이름)

인사("홍길동")
```

### 기본값 매개변수

```han
함수 인사(문자 이름 = "손님", 정수 횟수 = 1):
    반복 i 부터 1 까지 횟수:
        출력("안녕하세요, " + 이름)

인사()               // "안녕하세요, 손님"
인사("홍길동")       // "안녕하세요, 홍길동"
인사("홍길동", 3)    // 3번 출력
```

### 가변 매개변수

```han
함수 합계(정수... 수들):
    정수 합 = 0
    각각 수 안에 수들:
        합 = 합 + 수
    반환 합

합계(1, 2, 3)         // 6
합계(1, 2, 3, 4, 5)   // 15
```

### 다중 반환값

```han
함수 최대최소(배열 수들):
    반환 최대(수들), 최소(수들)

정수 큰값, 정수 작은값 = 최대최소([3, 1, 5, 2])
```

### 람다 (익명 함수)

```han
함수형 제곱 = (정수 x) => x * x

배열 숫자들 = [1, 2, 3, 4, 5]
배열 제곱들 = 숫자들.변환((정수 x) => x * x)
배열 짝수들 = 숫자들.걸러내기((정수 x) => x % 2 == 0)
```

## 7. 자료구조

### 배열

```han
배열 과일 = ["사과", "배", "감"]

과일[0]           // "사과"
과일.길이()       // 3
과일.추가("포도")
과일.제거("배")
과일.정렬()
과일.뒤집기()

// 고차 함수
과일.변환((문자 x) => 대문자(x))
과일.걸러내기((문자 x) => 포함(x, "사"))
과일.줄이기((문자 누적, 문자 값) => 누적 + "," + 값)
```

### 사전

```han
사전 사람 = {"이름": "홍길동", "나이": 30, "도시": "서울"}

사람["이름"]          // "홍길동"
사람["주소"] = "부산" // 추가
사람.키들()           // ["이름", "나이", "도시"]
사람.값들()           // ["홍길동", 30, "서울"]
사람.있음("나이")     // 참
사람.제거("도시")
```

### 목록 (구조체)

```han
목록 좌표:
    실수 x
    실수 y
    실수 z = 0.0    // 기본값

좌표 위치 = 좌표(1.0, 2.5)
좌표 위치 = 좌표(1.0, 2.5, 3.0)
위치.x    // 1.0
위치.y    // 2.5
```

### 열거형

```han
열거 방향:
    위
    아래
    왼쪽
    오른쪽

방향 이동 = 방향.위

선택 이동:
    경우 방향.위:
        출력("위로 이동")
    경우 방향.아래:
        출력("아래로 이동")
```

## 8. 객체 지향

### 객체 정의

```han
객체 사람:
    문자 이름
    정수 나이
    문자 직업 = "미정"    // 기본값

    함수 생성(자신, 문자 이름, 정수 나이):
        자신.이름 = 이름
        자신.나이  = 나이

    정의 소개(자신):
        출력(자신.이름 + " / " + 자신.나이 + "살")

    함수 나이증가(자신, 정수 증가량 = 1):
        반환 자신.나이 + 증가량
```

### 객체 생성 및 사용

```han
홍길동 = 사람("홍길동", 30)
홍길동.소개()
내년나이 = 홍길동.나이증가()
```

### 상속

```han
객체 학생 이어받기 사람:
    문자 학교
    정수 학년

    함수 생성(자신, 문자 이름, 정수 나이, 문자 학교, 정수 학년):
        부모.생성(이름, 나이)    // 부모 생성자 호출
        자신.학교 = 학교
        자신.학년 = 학년

    정의 소개(자신):            // 메서드 재정의
        부모.소개(자신)
        출력(자신.학교 + " " + 자신.학년 + "학년")

김철수 = 학생("김철수", 17, "한글고", 2)
김철수.소개()
```

### 인터페이스

```han
틀 도형:
    함수 넓이()
    함수 둘레()
    정의 정보출력()

객체 원 이어받기 도형:
    실수 반지름

    함수 생성(자신, 실수 반지름):
        자신.반지름 = 반지름

    함수 넓이(자신):
        반환 파이 * 자신.반지름 ** 2

    함수 둘레(자신):
        반환 2 * 파이 * 자신.반지름

    정의 정보출력(자신):
        출력("원: 반지름=" + 자신.반지름)
```

## 12. 계약 시스템

계약 없는 파일은 컴파일이 거부됩니다. 4가지 계약 중 하나 이상이 반드시 있어야 합니다.

### 헌법 — 전역 최상위 계약

```han
헌법 시스템안전:
    법령:  온도 < 85.0 그리고 전압 < 5.5
    법위반: 긴급정지()
헌법끝
```

### 법률 — 파일 전체 계약

```han
법률 데이터검증:
    법령:  입력값 아니다 없음
    법위반: 오류발생("입력 없음")
법률끝
```

### 규정 — 객체 범위 계약

```han
규정 센서:
    법령:  자신.연결됨 == 참
    법위반: 자신.재연결()
규정끝
```

### 법령 / 법위반 — 함수 단위 계약

```han
함수 나누기(실수 가, 실수 나):
    법령:  나 != 0.0
    법위반: 오류발생("0으로 나눌 수 없습니다")
    반환 가 / 나

함수 설정(정수 값):
    법령:  0 <= 값 그리고 값 <= 100
    법위반 경고: "값이 범위를 벗어났습니다"
    // 법위반 제재: 경고 / 보고 / 중단 / 회귀 / 대체
```

### 제재 종류

| 제재 | 동작 |
|------|------|
| `경고` | 경고 출력 후 계속 실행 |
| `보고` | 로그 기록 후 계속 실행 |
| `중단` | 오류 발생 후 실행 중단 |
| `회귀` | 복원지점으로 상태 복원 |
| `대체` | 대체값 또는 대체 함수로 교체 |

### 복원지점

```han
복원지점 안전상태
// ... 작업 ...
만약 오류발생:
    회귀 안전상태
```

## 13. 인터럽트 시스템

### OS 시그널

```han
신호받기 중단신호:
    출력("Ctrl+C 감지 — 정리 중")
    정리()
신호받기끝

신호무시 연결신호      // SIGPIPE 무시
신호기본 자식신호      // SIGCHLD 기본 처리
신호보내기 중단신호, PID   // 특정 프로세스에 신호 전송

// 신호 이름
// 중단신호  종료신호  강제종료  자식신호
// 사용자신호1  사용자신호2  연결신호  경보신호
// 중지신호  재개신호
```

### 하드웨어 인터럽트 (ISR)

```han
간섭 시간0넘침:
    카운터 = 카운터 + 1
간섭끝

간섭잠금    // 인터럽트 비활성화 (cli)
    // 임계 구역
```

## 17. 가속기 블록

TPU → NPU → GPU → CPU 순으로 자동 폴백합니다.

```han
// 자동 감지
가속기:
    행렬곱(A, B) => 결과
가속기끝

// 가속기 종류 지정
가속기 GPU:
    행렬곱(A, B) => 중간
    활성화(중간) => 결과
가속기끝

가속기 NPU:
    합성곱(이미지, 커널) => 특징
가속기끝

가속기 CPU:
    행렬합(A, B) => 합계
가속기끝
```

### 가속기 내장 연산

| 명령어 | 설명 |
|--------|------|
| `행렬곱(A, B) => 결과` | 행렬 곱 (GEMM) |
| `행렬합(A, B) => 결과` | 원소별 합 (Element-wise Add) |
| `합성곱(입력, 커널) => 결과` | 합성곱 (Convolution) |
| `활성화(입력) => 결과` | 활성화 함수 (ReLU) |
| `전치(A) => 결과` | 전치 (Transpose) |
