# 시작하기 (Getting Started)

## 1. 사전 요구사항

| 도구 | 버전 | 필수 여부 |
|------|------|---------|
| GCC | 9.0 이상 | ✅ 필수 |
| Make | 3.8 이상 | ✅ 필수 |
| LLVM / Clang | 14.0 이상 | 선택 (LLVM 백엔드) |
| Emscripten | 최신 | 선택 (WASM 빌드) |

### Linux (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install gcc make
# LLVM 백엔드 사용 시 (선택)
sudo apt install llvm-dev clang
```

### macOS

```bash
brew install gcc make
# LLVM 백엔드 사용 시 (선택)
brew install llvm
```

### Windows

MinGW-w64 설치 후 사용합니다.  
LLVM 백엔드는 WSL2 사용을 권장합니다.


## 3. 빌드

### 기본 빌드 (C 생성기 + 바이트코드 VM)

```bash
make
```

빌드 결과물:

| 파일 | 설명 |
|------|------|
| `kcode_gen` | C 코드 생성기 |
| `kbc` | 바이트코드 VM 실행기 |

### LLVM 백엔드 포함 빌드

```bash
make LLVM=1
```

추가 결과물: `kcode_llvm`

### 빌드 옵션 전체

```bash
make              # 전체 빌드 (gen + kvm + test)
make gen          # C 코드 생성기만
make kvm          # 바이트코드 VM만
make test         # 렉서 단위 테스트
make LLVM=1       # LLVM 백엔드 포함
make llvm-check   # LLVM 설치 여부 확인
make clean        # 빌드 결과물 삭제
make help         # 도움말
```

## 4. 첫 번째 프로그램

### hello.han 파일 작성

```han
정의 인사(문자 이름):
    출력("안녕하세요, " + 이름 + "!")

인사("세계")
```

### 실행 방법 3가지

**방법 1 — 바이트코드 VM (권장)**

```bash
./kbc exec hello.han
```

**방법 2 — C 코드로 컴파일 후 실행**

```bash
./kcode_gen hello.han -o hello.c
gcc -o hello hello.c -lm
./hello
```

**방법 3 — LLVM IR 생성 후 빌드**

```bash
./kcode_llvm hello.han -b -o hello
./hello
```

### 예상 출력

```
안녕하세요, 세계!
```

|--------|------|
| `.han` | Kcode 소스 파일 |
| `.kbc` | 바이트코드 컴파일 결과 |
| `.hg` | Kcode 헤더 파일 |

---

## 6. kbc 명령어 모음

```bash
./kbc exec   소스.han      # 컴파일 + 즉시 실행
./kbc compile 소스.han     # → 소스.kbc 생성
./kbc run    소스.kbc      # 바이트코드 실행
./kbc dump   소스.kbc      # 바이트코드 디스어셈블
```

## 7. kserver — 웹 IDE 연동

로컬에서 웹 IDE와 연동하는 HTTP 서버입니다.

```bash
# 빌드
gcc -O2 -o kserver kserver.c

# 실행 (기본 포트 8080)
./kserver

# 옵션
./kserver 3000              # 포트 지정
./kserver 8080 -llvm        # LLVM 컴파일러 사용
./kserver 8080 -cgen        # C 코드 생성기 사용
```

서버 동작 확인:

```bash
curl http://localhost:8080/health
```
