# 언어 레퍼런스 (Language Reference)

## 1. 설계 철학

### 한글 키워드 전용
모든 예약어를 한글로 작성합니다. 가능한 한 3자 이내로 설계했습니다.

```han
// ✅ Kcode 스타일
만약 온도 > 100:
    출력("끓는 중")

// ❌ 기존 언어 방식 (Kcode에서 사용 불가)
if temperature > 100:
    print("boiling")
```

### 들여쓰기 기반 블록 (Python 스타일)
공백 4칸 또는 탭 1개를 블록 구분에 사용합니다.

### 포인터 없음
GC(가비지 컬렉터)가 메모리를 자동 관리합니다.

### 함수와 정의 구분
반환값 유무로 두 키워드를 명확히 구분합니다.

```han
함수 더하기(정수 가, 정수 나):   // 반환값 있음
    반환 가 + 나

정의 인사(문자 이름):            // 반환값 없음 (void)
    출력("안녕하세요, " + 이름)
```

## 2. 자료형

| 키워드 | 설명 | 예시 |
|--------|------|------|
| `정수` | 소수점 없는 숫자 | `1`, `-5`, `100` |
| `실수` | 소수점 숫자 | `3.14`, `-0.5` |
| `글자` | 문자 하나 (char) | `'가'`, `'A'` |
| `문자` | 문자열 (string) | `"안녕하세요"` |
| `논리` | 참 또는 거짓 | `참`, `거짓` |
| `없음` | null | — |
| `2진` | 2진법 | `0b1010` |
| `8진` | 8진법 | `0o17` |
| `16진` | 16진법 | `0xFF` |
| `행렬` | 2차원 배열 (AI용) | `[[0.1, 0.2], [0.3, 0.4]]` |
| `텐서` | N차원 텐서 | `텐서([1.0, 2.0], [2])` |
| `사진` | 파일 이미지 | `사진열기("photo.jpg")` |
| `그림` | 코드로 그리는 캔버스 | `그림만들기(800, 600)` |
| `함수형` | 람다 / 익명 함수 | `(x) => x * x` |

## 5. 제어 구조

### 조건문

```han
만약 점수 >= 90:
    출력("A 등급")
아니면 만약 점수 >= 80:
    출력("B 등급")
아니면:
    출력("C 이하")
```

### 반복문

```han
// 범위 반복
반복 i 안에 범위(0, 10):
    출력(i)

// 컬렉션 반복
각각 항목 안에 목록:
    출력(항목)

// 조건 반복
동안 카운터 < 5:
    카운터 += 1

// 무한 반복
동안 참:
    // ...
    멈춤   // break
```

### 반복 제어

```han
멈춤        // break
계속        // continue
반환 값     // return
```

## 6. 함수

```han
// 기본 함수
함수 합계(정수 가, 정수 나):
    반환 가 + 나

// 기본값 매개변수
함수 인사(문자 이름, 문자 말 = "안녕하세요"):
    반환 말 + ", " + 이름

// 가변 매개변수
함수 모두더하기(정수... 숫자들):
    정수 합 = 0
    각각 n 안에 숫자들:
        합 += n
    반환 합

// 람다 (익명 함수)
함수형 제곱 = (정수 x) => x * x
```

## 7. 자료구조

### 배열

```han
배열 과일 = ["사과", "배", "감"]
과일[0]              // 접근
과일.추가("포도")    // 추가
과일.제거("배")      // 제거
과일.길이()          // 길이
과일.정렬()          // 정렬
```

### 사전

```han
사전 사람 = {"이름": "홍길동", "나이": 30}
사람["이름"]         // 접근
사람["주소"] = "서울"  // 추가/수정
```

### 구조체 (목록)

```han
목록 좌표:
    실수 x
    실수 y

좌표 위치 = 좌표(1.0, 2.5)
출력(위치.x)
```

### 객체 (클래스)

```han
객체 사람:
    글자 이름
    정수 나이

    함수 생성(자신, 문자 이름, 정수 나이):
        자신.이름 = 이름
        자신.나이 = 나이

    정의 소개(자신):
        출력(자신.이름 + " / " + 자신.나이 + "살")

// 상속
객체 학생 이어받기 사람:
    문자 학교

    함수 생성(자신, 문자 이름, 정수 나이, 문자 학교):
        부모.생성(이름, 나이)
        자신.학교 = 학교
```

### 열거형

```han
열거 방향:
    위
    아래
    왼쪽
    오른쪽

방향 이동방향 = 방향.위
```

### 인터페이스

```han
틀 도형:
    함수 넓이()
    함수 둘레()
    정의 정보출력()

객체 원 이어받기 도형:
    // 넓이(), 둘레(), 정보출력() 모두 구현 필수
    // 미구현 시 컴파일 오류
```

## 9. 내장 함수

### 출력

```han
출력("텍스트")
출력(변수)
출력("값: " + 변수)
```

### 수학

```han
제곱근(16.0)       // 4.0
절댓값(-5)         // 5
최대(3, 7)         // 7
최소(3, 7)         // 3
반올림(3.7)        // 4
올림(3.2)          // 4
내림(3.9)          // 3
난수정수(1, 10)    // 1~10 랜덤 정수
```

### 문자열

```han
길이("안녕")         // 2
대문자("hello")      // "HELLO"
소문자("HELLO")      // "hello"
자르기("안녕하세요", 0, 2)   // "안녕"
찾기("안녕하세요", "하세")   // 2 (시작 인덱스)
바꾸기("Hello World", "World", "Kcode")
나누기("a,b,c", ",")         // ["a", "b", "c"]
합치기(["a", "b", "c"], "-") // "a-b-c"
```

### 형변환

```han
정수로("42")     // 42
실수로("3.14")   // 3.14
문자로(42)       // "42"
```

### 파일

```han
파일전체읽기("경로/파일.txt")
파일전체쓰기("경로/파일.txt", 내용)
파일있음("경로/파일.txt")   // 논리 반환
파일목록("./폴더")          // 배열 반환
파일이름("경로/파일.txt")   // "파일.txt"
파일확장자("파일.txt")      // ".txt"
```

## 10. 계약 시스템 (Contract System)

코드 실행 안전성을 보장하는 조건 검증 시스템입니다.

```han
헌법:
    // 절대 지켜야 할 조건 (위반 시 즉시 종료)
    법령 나이 >= 0, 중단
    법령 이름 != "", 중단
헌법끝

법률:
    // 함수 진입 전 조건 (사전 조건)
    법령 점수 >= 0 그리고 점수 <= 100, 중단
법률끝

규정:
    // 위반 시 처리 방식 설정
    법위반 데이터없음:
        출력("데이터가 없습니다")
        복원
규정끝
```
